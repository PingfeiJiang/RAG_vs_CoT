Key Highlights

  • Electronics are Sealed for water and dust protection (IP65 rating).
  • Wheelchair turning_radius is 360 degrees.
  • Battery capacity Minimum 20 miles range.
  • Wheelchair meets Relevant safety standards (e.g., ISO 7176).
Requirement Features & Specifications
Accessibility and Adaptability
  • Seat is Height adjustable to accommodate different users.
  • Wheelchair supports Attachment of assistive devices (e.g., oxygen tanks, ventilators).
  • Control System is Configurable for left- or right-handed users.
  • Wheelchair is designed for Use on various terrains (e.g., sidewalks, ramps, grass).
Durability and Weather Resistance
  • Frame is Powder-coated for corrosion resistance.
  • Electronics are Sealed for water and dust protection (IP65 rating).
  • Upholstery is Water-resistant and easy to clean.
  • Wheelchair is designed for All-weather operation.
Enhanced Maneuverability in Tight Spaces
  • Wheelchair has Omni-directional wheels.
  • Wheelchair turning_radius is 360 degrees.
  • Control System features Precise joystick control.
  • Motor type High-torque DC motors.
Extended Battery Life and Efficient Power Management
  • Battery is Lithium-ion with high energy density.
  • Battery Management System (BMS) provides Real-time monitoring and optimization.
  • Motor Controller uses Regenerative braking system.
  • Wheelchair has Sleep mode for power saving.
  • Battery capacity Minimum 20 miles range.
Improved User Comfort and Ergonomics
  • Seat is made of Breathable memory foam.
  • Backrest is Adjustable for lumbar support.
  • Armrests are Height and width adjustable.
  • Footrests are Angle adjustable.
  • Wheelchair includes Suspension system for shock absorption.
Intuitive and Customizable Control Interface
  • Control Panel features LCD screen with user-friendly interface.
  • Control System allows Customizable speed and acceleration profiles.
  • Control System supports Multiple control modes (e.g., joystick, head control, sip-and-puff).
  • Control System has Emergency stop button.
  • Control System integrates Bluetooth connectivity for smartphone control and diagnostics.
Portability and Ease of Transportation
  • Wheelchair is Foldable for compact storage.
  • Frame is Lightweight for easy lifting.
  • Battery is Removable for separate charging and transport.
  • Wheelchair includes Integrated carrying handles.
Safe and Reliable Operation
  • Wheelchair has Anti-tip wheels.
  • Wheelchair includes Automatic braking system on inclines.
  • Wheelchair is equipped with LED lighting for visibility.
  • Wheelchair meets Relevant safety standards (e.g., ISO 7176).
  • Frame is made of Lightweight, high-strength aluminum alloy.
  • Wheelchair has Audible alerts for low battery and system errors.
Smart Features and Connectivity
  • Wheelchair integrates GPS tracking for location monitoring.
  • Control System supports Voice control functionality.
  • Wheelchair connects to Smart home devices via IoT.
  • Wheelchair provides Fall detection and emergency alerts.
